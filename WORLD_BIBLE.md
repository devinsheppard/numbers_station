# WORLD BIBLE

## Project Boundary

World implementation is not part of Milestone 005.

This document is reserved for locations, factions, timeline, environmental rules, symbolic motifs, and continuity constraints.

No world canon has been established yet.

## Geographic and privacy constraints

The game is set in the real Irish Wilderness and surrounding rural region of
south-central Missouri. Future work should preserve authentic terrain,
forests, waterways, topography, road relationships, and travel distances.

Real homes, businesses, and identifiable private properties must not be
recreated. Buildings, businesses, structures, and story locations will be
fictional while fitting naturally into the real landscape.
